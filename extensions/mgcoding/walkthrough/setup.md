## 1. Configura un modello

Scegli dove far girare l'agente: un modello **locale** (Ollama, gratis e offline) o un
servizio **cloud** (Gemini, ChatGPT, Claude, Azure, OpenRouter).

Apri la **Configurazione guidata**, scegli il servizio e — se cloud — incolla la API key.

> Suggerimento: per Google Gemini serve la API key gratuita di **AI Studio**.
